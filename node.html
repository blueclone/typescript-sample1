node

node.jsのnpmコマンド

$ npm init
「現在のディレクトリ下はnpmが管理する」という宣言
$ npm install パッケージ名
npmのリポジトリからパッケージをダウンロードしてくるコマンド
$ npm run hogehoge
package.jsonに書かれたシェルスクリプトを実行するコマンド
$ npm install hogehoge
package.json にローカルインストールしたい（= そのディレクトリだけで使用できるようにしたい）パッケージ（例 : cowsay）を追加するとき（本番環境でも使用するパッケージ）
$ cat hogehoge
シェルからファイル名を一つ指定して実行すると、指定されたファイルの内容が画面に表示される

使い方一例

$ npm run dev
package.jsonのdevを実行する
