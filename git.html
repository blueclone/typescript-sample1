gitのコマンド

$ git add .
変更されたすべてのファイルをコミットするためのステージへ移動
git add hogehoge
変更された hogehoge というファイルをコミットするためのステージへ移動
$ git commit -m "コメント"
コメントを記述して commit する
$ git commit --amend -m "新たなコメント"
直前に commit したファイルのコメントを新たなコメントに修正する
$ git push origin カレントブランチ名
カレントブランチを同名のリモートブランチにプッシュする
$ git push origin HEAD
カレントブランチ名を指定しなくても、同名のリモートブランチにカレントブランチをプッシュできる
$ git log
コミットログを表示
