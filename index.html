Viteとは

Vite（フランス語で「素早い」という意味の単語で /vit/ ヴィートのように発音）は、現代の Web プロジェクトのために、より速く無駄のない開発体験を提供することを目的としたビルドツールです。2 つの主要な部分で構成されています:

gitについて

  1. commitはタスク単位でやろう
    なぜか？
    大きな作業単位だと、あとで見返す時の負担が重くなるから
  2. commitしたら、すぐpushしよう
    なぜか？
    もしPCが壊れたとしても、リモートに上げているので作業の回復の労力が少なくなるので

よく使うgitのコマンド:add, checkout(またはswitch), clone, commit, fetch, merge, pull, push

その次に使うgitのコマンド:reset rebase cherry-pick

git add . : 変更からステージへ全てを持っていくコマンド

git commit -m "内容" : コミットする内容を記述する

$ git push origin HEAD : カレントブランチをリモートの同名ブランチにプッシュするための便利な方法

npm run dev : vite + typescriptを更新する

reset.cssの意味を調べ npm package でインストールする

aタグの使い方 : 別タブで開きたい場合、target="_blank" と rel="noopener noreferrer" をaタグに記述する。

zinit : zsh の便利なプラグインを管理するツール。 zinitインストール

CSSの詳細度 : ある要素に最も関連性の高い CSS 宣言を決定するためにブラウザーが使用するアルゴリズム

dl・dt・ddタグ : 説明リストタグといい、「用語の説明」や「Q&A」「お知らせ(日付とタイトル)」などでよく用いられる、HTMLタグ

カルーセルUI : 複数の選択肢やコンテンツを列に表示する場合に利用する。ぜひ使ってみよう。
