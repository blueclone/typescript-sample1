viteの備忘録

vite

Viteとは
現代のWebプロジェクトのために、より速く無駄のない開発体験を提供することを目的としたビルドツールです。2 つの主要な部分で構成されています

vite で新規プロジェクトの始め方

$ node -v
Node.js のバージョン確認
$ npm -v
npmのバージョンの確認
$ npm update -g npm
npm のアップデート
$ npm create vite
vite の新規プロジェクトを作る
$ npm install
新規 or 更新された package.json を元に node_modules を作成するとき
$ npm run dev
package.jsonに書かれたシェルスクリプト(今回は dev )を実行するコマンド
