aタグの使い方
別タブで開きたい場合、target="_blank" と rel="noopener noreferrer" をaタグに記述する。
dl・dt・ddタグ
説明リストタグといい、「用語の説明」や「Q&A」「お知らせ(日付とタイトル)」などでよく用いられる
input : 入力欄（フォーム入力）要素 - HTML | MDN
ユーザーからデータを受け取るための、ウェブベースのフォーム用の操作可能なコントロールを作成する
label : 入力欄ラベル要素 - HTML | MDN
label タグを input 要素と関連付けると、いくらかの利点が発生する
